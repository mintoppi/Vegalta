# ベガルタ仙台 非公式ファンサイト 設計書

## 1. プロジェクト概要
ベガルタ仙台を応援するための、モダンでダイナミックな非公式ファンサイトです。
ファンが最新情報を確認し、次の試合への期待感を高めることができるプラットフォームを提供します。

## 2. 技術スタック
- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)
- **Fonts**: 
  - Barlow Condensed (英字・見出し用)
  - Noto Sans JP (日本語・本文用)
- **デザイン手法**: 
  - CSS Variables（変数）による一貫した配色管理
  - Glassmorphism（透過背景）の採用
  - Intersection Observer APIによるスクロールアニメーション

## 3. デザインシステム
### 3.1 カラーパレット
- **メインゴールド**: `#D4AF37` (旗やロゴ等に使用)
- **ブライトゴールド**: `#FCC800` (アクセント、ボタン、重要な見出し)
- **ベガルタブルー**: `#004D96`
- **ダークブルー**: `#002B55` (背景のグラデーション、カード等)
- **ダーク背景**: `#111111` (全体のベースカラー)

### 3.2 タイポグラフィ
- 見出し (`h1`〜`h6`): Barlow Condensed, 大文字化, レタースペーシングあり
- 本文: Noto Sans JP

## 4. コンポーネント構成
### 4.1 ヘッダー (Main Header)
- 固定（Fixed）配置、背景はグラスモーフィズム対応。
- ロゴ（VEGALTA SENDAI）とナビゲーションリンク。

### 4.2 ヒーローセクション (Hero Section)
- 全画面表示のインパクトのあるセクション。
- スタジアムの背景画像（`assets/hero_bg.png`）を配置。
- ブランドメッセージ「VEGALTA GOLD SPIRIT」を表示。

### 4.3 次試合情報 (Next Battle)
- カウントダウンタイマー機能を搭載。
- 次回の対戦カードとスタジアム情報を表示。
- スタイリッシュなカードデザイン。

### 4.4 注目選手紹介 (Player Showcase) - Pokemon Card Style
ベガルタ仙台の選手11名を、トレーディングカードゲーム（ポケモンカード）をモチーフにしたリッチなUIで表示。

- **カード内部構造**:
  - **ヘッダー**: 進化ステージ（ポジション特性）、選手名、HP（身長数値等）。
  - **画像フレーム**: メタリックゴールドの境界線と、公式2025プロフィール写真。
  - **スペック**: 身長・体重・出身地を一行で表示。
  - **ボディ（技）**: プレイスタイルに基づいた2つのユニークな「ワザ」名・ダメージ量・説明文・エネルギーコスト（属性アイコン）。
  - **フッター**: ２０２５年度の出場数、出場時間、ゴール数、好きなアーティスト、よく聴く音楽をhttps://www.vegalta.co.jp/team/top-{number}.htmlから取得して表示
  
- **ビジュアル・技術仕様**:
  - **3Dホバーエフェクト**: マウスホバー時にカードが3D空間でリアルに傾き、影が連動する演出。
  - **ホログラフィック・グリッター**: 背景に動くグラデーションを重ね、レアカードのような輝きを再現。
  - **グラスモーフィズム**: 半透明背景と `backdrop-filter` を使用。
  - **SVGアイコン**: エネルギー部分に属性カラーの円形アイコンを使用。

- **データ連携**: `script.js` 内の `players` 配列から動的に生成。公式写真URLは選手番号に基づき自動生成。

### 4.5 ニュースセクション (最新のX/Twitter投稿)
- 公式X（@vega_official_）の最新ポストをインデックスとして表示。
- 移籍情報や公式チャンネル更新などの最新トピックを配置し、詳細へのリンク（ソース）を提供。

### 4.6 過去の試合結果 (Match Archive) - Gorgeous Edition
- 2025シーズンの最終5試合を公式データから取得して表示。
- **ゴージャスなUI**: ネオンエフェクト、グラスモーフィズム、ホバー時のトランスフォームアニメーションを採用。
- 勝敗に応じた境界線カラーとグロー効果による視覚的強調。

## 5. 動的機能
- **ポケモンカードレンダリング**: 注目選手（林、鎌田、相良、荒木、宮崎ら計11名）をカード形式で動的に描画。公式プロフィール画像と連動。
- **リアルタイム秒単位カウントダウン**: 2026年開幕戦に向けた、毎秒更新される高精度タイマー。
- **最新 X ポスト取得 (RSS2JSON)**: Nitter経由で公式アカウントの最新情報を動的にフェッチ。
- **グローバル・スクロールアニメーション**: Intersection Observerを使用した、セクションごとの浮き上がり/フェードイン効果。
- **マルチデバイス対応**: メディアクエリによるモバイル最適化。全セクションがスマートフォン閲覧に適した縦並びレイアウトへ自動調整される。
