# ベガルタ仙台 非公式ファンサイト 設計書

## 1. プロジェクト概要
ベガルタ仙台を応援するための、モダンでダイナミックな非公式ファンサイトです。
ファンが最新情報を確認し、次の試合への期待感を高めることができるプラットフォームを提供します。

## 2. 技術スタック
- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)
- **Fonts**: 
  - Barlow Condensed (英字・見出し用)
  - Noto Sans JP (日本語・本文用)
- **デザイン手法**: 
  - CSS Variables（変数）による一貫した配色管理
  - Glassmorphism（透過背景）の採用
  - Intersection Observer APIによるスクロールアニメーション

## 3. デザインシステム
### 3.1 カラーパレット
- **メインゴールド**: `#D4AF37` (旗やロゴ等に使用)
- **ブライトゴールド**: `#FCC800` (アクセント、ボタン、重要な見出し)
- **ベガルタブルー**: `#004D96`
- **ダークブルー**: `#002B55` (背景のグラデーション、カード等)
- **ダーク背景**: `#111111` (全体のベースカラー)

### 3.2 タイポグラフィ
- 見出し (`h1`〜`h6`): Barlow Condensed, 大文字化, レタースペーシングあり
- 本文: Noto Sans JP

## 4. コンポーネント構成
### 4.1 ヘッダー (Main Header)
- 固定（Fixed）配置、背景はグラスモーフィズム対応。
- ロゴ（VEGALTA SENDAI）とナビゲーションリンク。

### 4.2 ヒーローセクション (Hero Section)
- 全画面表示のインパクトのあるセクション。
- スタジアムの背景画像（`assets/hero_bg.png`）を配置。
- ブランドメッセージ「VEGALTA GOLD SPIRIT」を表示。

### 4.3 次試合情報 (Next Battle)
- カウントダウンタイマー機能を搭載。
- 次回の対戦カードとスタジアム情報を表示。
- スタイリッシュなカードデザイン。

### 4.4 選手一覧 (Meet the Squad)
- 選手データをJavaScriptで管理し、動的に生成。
- **ポケモンカード風デザイン**: HP（背番号由来）、技（スキル）、属性、ホログラフィック効果、3D回転アニメーションを採用。
- 公式サイトの情報を反映（相良竜之介選手など）。

### 4.5 ニュースセクション (最新のX/Twitter投稿)
- 公式X（@vega_official_）の最新ポストをインデックスとして表示。
- 移籍情報や公式チャンネル更新などの最新トピックを配置し、詳細へのリンク（ソース）を提供。

### 4.6 過去の試合結果 (Match Archive)
- 2025シーズンの直近試合結果を表示。
- スコア、対戦相手、日付を表示し、勝敗に応じた色分け（Win/Draw/Loss）による視認性の向上。

## 5. 動的機能
- **ポケモンカードレンダリング**: 相良選手などのプロフィールをカード形式で動的に描画。
- **2026年開幕カウントダウン**: 次回シーズンの開幕に向けたカウントダウンタイマー。
- **スクロールアニメーション**: Intersection Observer APIを使用したフェードイン効果。
- **レスポンシブ対応**: モバイル環境ではナビゲーションを非表示にし、タイトルサイズやカードレイアウトを最適化。
